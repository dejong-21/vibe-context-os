# Demo Workspace

This small workspace demonstrates Vibe Coding Context OS without private sessions or local user paths.

## Project

The demo project is a tiny AI coding workflow repository. It has agent rules, package scripts, a source file, and a public release checklist.

## Verification

```bash
npm run lint
npm run build
npm run smoke
```

## AI Workflow

- Keep generated agent rules under review before applying them.
- Use task packs for cross-agent handoff.
- Run privacy audit before copying output into a public README or GitHub profile.
