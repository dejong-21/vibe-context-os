## Vibe Coding Context OS

I am building a local-first context engineering tool for AI-assisted development. It scans project rules, docs, code, and optional Codex sessions, then generates task packs, agent instructions, MCP tools, privacy gates, drift reports, and public-safe portfolio artifacts.

Current focus: `Context Engineering` `AI Coding Tools` `Debugging And QA` `Docs And Knowledge`.

What it demonstrates:

- Local-first LLM application architecture
- Context engineering for AI coding agents
- MCP stdio server design
- Multi-agent exports for Codex, Claude Code, Cursor, Gemini CLI, Cline/Roo, Continue, and GitHub Copilot
- Secret redaction, privacy audit, and publish gates
- CI-backed packaging and installed-package smoke tests
